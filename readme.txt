# IP Location Tracker

This is a simple Python program that tracks the approximate location of a device based on its public IP address. The location data is fetched using the IP address and provides details like city, region, country, latitude, and longitude.

## Features

- Retrieves the public IP address of the device.
- Uses IP-based geolocation to estimate the location of the device.
- Displays location details such as city, region, country, and coordinates (latitude and longitude).

## Requirements

- Python 3.x
- `requests` library (for making HTTP requests)
